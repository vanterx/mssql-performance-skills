# sqlmigration-review References

## When to Load This Reference

Load `check-explanations.md` when:

- A user asks "explain check Y##" or "what does Y## mean"
- A user asks for deeper fix options beyond what the SKILL.md Fix line provides
- A user asks about a specific category ("explain all the Azure SQL checks")
- A user wants to understand the threshold rationale for a specific check
- A user asks about related checks for a finding that fired
